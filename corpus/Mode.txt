Trivia[[280]edit]

     * In the Japanese version of Melee, the Main Menu is instead named Top Menu. Additionally, it [281]was originally going to be named Top Menu in English as well, but the final version of the game reverts the name back to Main Menu.
     * Ultimate uses the name Top Menu in English, under the Dashboard's "Back to Top Menu" option.
