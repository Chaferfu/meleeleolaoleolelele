   Welcome to the wiki! The Community Portal is where this wiki community comes together to organize and discuss projects for the wiki. To see the most recent discussions, click the Discussion tab above.

   You can find out more general information about the wiki on the [7]About page.

New to wikis?[[15]edit]

   If you are new to wikis, you may want to read [16]this page first which will teach you the basics of SmashWiki and other helpful information.

How to help[[17]edit]

     * If you want to help but you're not sure where to start, try improving the various [18]stub articles by adding content.
     * You can find a list of useful templates on [19]Category:Templates, some of which are documented on [20]SmashWiki:Templates.
     * A style guide for this wiki can be found on [21]SmashWiki:Manual of Style.
     * If you need help editing, start by reading [22]Help:Editing.

Who's here?[[23]edit]

     * To follow the changes to this wiki, and to look out for new users and edits use [24]recent changes.

How to discuss stuff[[25]edit]

     * If you want to discuss anything specific to an article, you can use [26]the talk pages for that article
     * For more general discussions you can use the Discussion tab of this page.

   Useful activities

     * Help expand some [47]article stubs.
     * [48]Locate and [49]mark pages that need expanding as [50]article stubs.
     * Create links to some [51]orphaned pages.
     * Upload and add some images! Visit [52]Special:Upload. Add them to a category too, like [[Category:Images]].
     * Write a new article that's on the [53]the wanted list!
     * Consider splitting up very [54]long pages.
     * Create some links on [55]dead end pages.
     * Add more...
